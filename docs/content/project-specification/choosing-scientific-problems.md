# How to choose a good scientific problem

Core characteristics of a good M.Sc. thesis problem:

* Fits personal preference/interests
* Challenging for the student
* Creates new experiences and contacts for the student
* Rich amount of scientific literature
* Few external dependencies
* Feedback must be rapid (is the project on the right track)
* Equilibrates feasibility vs. interest
* Collected data allow for statistical analyses
