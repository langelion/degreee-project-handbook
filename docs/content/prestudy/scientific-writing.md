# Scientific writing

Man lär sig både genom att få och ge respons på en text under skrivprocessens gång.

Vi får insyn i och kunskap om den verkliga SKRIVPROCESSEN, och att den är
mycket icke-linjär.

Utkast ; copy/paste ; skriv ; skriv om ;
flytta ; omformulera ; förändra ; förkasta ; börja om ; ta tillbaka ;

| ALLA kämpar med sin text.
| Det finns i princip INGEN som skriver perfekt från början.

"Övning, arbete, är här som på varje område, den enda säkra metoden"
