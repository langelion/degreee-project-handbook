# Presentation

## Procedure

* Academic supervisors approves that the report is ready for presentation.
* Examiner approves the report is ready for presentation.
* Note: the report does not have to be 100% ready.
* Apply for presentation. No later than the first day of the week before the presenation.
Add schedule/scheme.
* The degree project coordinator and the examiner decides on the final date.
* The degree project coordinator decides the opponent.
